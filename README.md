# Selenium Cucumber Automation Framework

## Tech Stack
- Java 11
- Selenium 4
- Cucumber BDD
- JUnit 5
- Maven

## How to Run Tests
mvn clean test

## Framework Features
- Page Object Model
- Explicit Waits
- Cucumber BDD
- Maven build
- JUnit Assertions

## Reports
- Cucumber default reports
